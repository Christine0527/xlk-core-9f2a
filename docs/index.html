主要功能

📍

一鍵移動定位

在地圖上點任意位置，iPhone GPS 立即更新，無需越獄。

🗺️

路線規劃 & 自動行走

設定多個路線點，模擬真實移動速度，步行 / 騎車 / 開車皆支援。

🕹️

搖桿即時控制

畫面搖桿拖曳即可連續移動，適合需要精細控制方向的使用情境。

🔒

無需越獄

透過 Apple 開發者功能連線，不需要越獄或修改系統，安全無風險。

🖥️

Mac & Windows

同時支援 macOS 與 Windows 10 / 11，跨平台都能使用。

⚡

支援 iOS 16 / 17+

自動偵測 iOS 版本，舊版直連，iOS 17+ 自動啟用進階通道。

使用需求

🖥️ 電腦

  • macOS 11+ 或 Windows 10/11
  • USB 連接埠
  • Windows 需安裝 iTunes

📱 iPhone

  • iOS 14 以上
  • USB 傳輸線連接電腦
